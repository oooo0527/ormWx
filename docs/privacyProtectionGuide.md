# 小程序用户隐私保护指引

## 1. 引言

感谢您使用我们的小程序。我们非常重视您的隐私和个人信息安全。本指引旨在帮助您了解我们会收集哪些数据、为什么收集这些数据以及如何使用和保护这些数据。

## 2. 我们如何收集和使用您的个人信息

为了向您提供更好的服务，我们可能会收集以下类型的个人信息：

### 2.1 您主动提供的信息
当您使用我们的服务时，我们会收集您主动提供的信息，例如：
- 您填写的昵称
- 您上传的头像
- 您发布的内容

### 2.2 设备信息
我们会收集您使用我们服务时所使用的设备信息，例如：
- 设备型号
- 操作系统版本
- 设备唯一标识符

### 2.3 日志信息
当您使用我们服务时，我们会自动收集某些信息并存储在服务器日志中，例如：
- IP地址
- 浏览器类型
- 访问日期和时间
- 页面浏览记录

## 3. 我们如何保护您的个人信息

我们采取各种安全措施来保护您的个人信息，防止未经授权的访问、使用或泄露。我们会定期审查我们的信息收集、存储和处理实践，以确保个人信息的安全。

## 4. 您的权利

您有权访问、更正、删除您的个人信息，以及撤回您之前授予的同意。如果您希望行使这些权利，请通过客服联系我们。
