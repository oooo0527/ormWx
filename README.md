# 明星互动平台微信小程序

一个包含明星档案、数据工坊、作品安利和粉丝心声四大模块的微信小程序，支持游客访问和注册登录两种访问形式。

## 功能模块

### 1. 明星档案
- 展示明星基本信息
- 查看明星详细档案
- 支持游客和登录用户访问

### 2. 数据工坊
- 展示各类数据报告
- 登录用户可上传数据
- 数据分类展示

### 3. 作品安利
- 展示影视作品信息
- 支持点赞和分享功能
- 作品详情展示

### 4. 粉丝心声
- 粉丝可发布心声（需登录）
- 支持点赞功能
- 心声列表展示

## 技术特点

- 基于微信小程序原生开发
- 支持游客访问和注册登录两种模式
- 响应式布局适配不同屏幕尺寸
- 数据本地存储（模拟）
- 用户友好的交互设计

## 使用说明

1. 将项目导入微信开发者工具
2. 选择对应的 appId 或使用测试号
3. 编译并预览小程序

## 注意事项

- 本项目为前端展示层，实际使用时需要配合后端服务
- 数据存储采用本地模拟方式，实际项目中应替换为真实 API 调用
- 图片资源需要根据实际需求替换
