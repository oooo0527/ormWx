# 明星互动平台微信小程序

一个包含明星档案、数据工坊、作品安利和粉丝心声四大模块的微信小程序，支持游客访问和注册登录两种访问形式。

## 功能模块

### 1. 明星档案
- 展示明星基本信息
- 查看明星详细档案
- 支持游客和登录用户访问

### 2. 数据工坊
- 展示各类数据报告
- 登录用户可上传数据
- 数据分类展示

### 3. 作品安利
- 展示影视作品信息
- 支持点赞和分享功能
- 作品详情展示

### 4. 粉丝心声
- 粉丝可发布心声（需登录）
- 支持点赞功能
- 心声列表展示

### 5. 消息通知
- 评论通知：当其他用户评论你的帖子时，会在消息中心收到通知
- 回复通知：当其他用户回复你的评论时，会在消息中心收到通知

## 技术特点

- 基于微信小程序原生开发
- 支持游客访问和注册登录两种模式
- 响应式布局适配不同屏幕尺寸
- 使用云开发实现后端功能
- 用户友好的交互设计

## 使用说明

1. 将项目导入微信开发者工具
2. 选择对应的 appId 或使用测试号
3. 部署云函数：
   - 右键点击`cloudfunctions`目录下的各个云函数
   - 选择"上传并部署：云端安装依赖"
4. 初始化数据库：
   - 在云开发控制台中调用`initDatabase`云函数的各个初始化方法
5. 编译并预览小程序

## 注意事项

- 本项目使用微信云开发作为后端服务
- 需要在微信公众平台开通云开发功能
- 图片资源需要根据实际需求替换
